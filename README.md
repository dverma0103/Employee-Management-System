# Employee Management System (Python CLI)

A menu-driven Python application to manage employee records using core Python concepts.

## Features
- Add employee
- View all employees
- Search employee by ID
- Update employee details
- Delete employee

## Concepts Used
- Python functions
- Lists & dictionaries
- Loops and conditionals
- Input validation
- Menu-driven program

1. Clone the repository
